# Thermostat---STM32L476RG-Nucleo
Thermostat using STM32L476RG Nucleo Chipboard


Original design and some code from repo https://github.com/controllerstech/STM32-HAL/tree/master/LCD1602_I2C_NOHAL

IDE: STM32CubeIDE.

Purpose: Learn Bare Metal Embedded Software.

Project: Create software for a STM32L476RG Nucleo chip board that replicates the functionallity of a Thermostat (read temperatures, display temperatures to a screen, call for heat/cool/fan).
